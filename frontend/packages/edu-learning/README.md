# @coze-edu/learning

教育平台 - 学生端学习模块（三阶段学习）

## 功能说明

支持三种渐进式学习模式：
- **剧本引导学习**：通过结构化剧本进行三阶段引导式学习
- **模板定制开发**：基于模板进行Bot定制化开发
- **自主Bot开发**：完全自主的AI Agent开发

## 核心页面

- `learning-center`: 学习中心（剧本库/模板库）
- `script-learning`: 剧本学习页面（三阶段）
- `template-learning`: 模板定制页面
- `bot-development`: 自主Bot开发页面
- `my-projects`: 我的学习项目
