## 6. 关键交互流程图
```mermaid
flowchart LR
    A[进入平台管理] --> B{选择Tab}
    B -->|计费管理| C[选择筛选条件]
    C --> D[请求后端聚合接口]
    D --> E[刷新指标卡/图表/列表]
    B -->|统计模块| F[选择统计维度]
    F --> G[请求统计接口]
    G --> H[刷新趋势图与排行表]
```

## 7. UI 风格约束
1. Tab、筛选器、标签样式与现有空间管理模块保持一致。
2. 指标卡数字字号大于普通文本一级，颜色遵循主题色层级。
3. 表格行高与现有发布管理列表统一，避免视觉割裂。
4. 空状态、错误状态使用项目统一组件，不新增独立视觉体系。
